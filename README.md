# PrimeNumberCalculator

Compute **1000 prime numbers with 512-bit length** and measure the **time taken** for the calculation.

This tool provides a simple and fast way to test CPU performance with large prime number calculations ‚Äì perfect for benchmarking and comparing different systems.
